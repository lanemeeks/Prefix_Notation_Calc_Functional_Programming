# Prefix_Notation_Calc_Functional_Programming
Current files and their roles:
devlog - development log of the work done for this project
prefix_calc - this is the file the user should run to use the functional prefix expression calculator
prefix_calc.rkt~ - this is a file that the Racket IDE on Mac creates to preserve the state of the racket source code immediately before it was modified.
Pesudocode - personal reference doc of how I would approach this problem imperatively

Instructions: Move to the project file within your terminal and run prefix_calc.rkt with racket and either a batch flag (-b or --batch) or not. You will be prompted to enter a prefix expression, do so or enter 'quit' to exit the program. You may also reference previous results using $id within the pre fix expression. The id of the value is provided when the result is given.
